# CalHasCulator

A sleek, single-file calculator web app built with HTML, CSS, and JavaScript — no frameworks, no build tools, no dependencies. Just open it in your browser and start calculating.

## Features

- ➕ Basic arithmetic operations (addition, subtraction, multiplication, division)
- ⚡ Instant calculations with a clean, responsive interface
- 📱 Works on desktop and mobile browsers
- 🔌 Fully offline — no internet connection required after downloading
- 📦 Single HTML file — easy to open, share, or embed anywhere

## Getting Started

No installation needed!

1. Download or clone this repository
2. Open `index.html` (or your calculator's file name) in any web browser
3. Start calculating

```bash
git clone https://github.com/your-username/CalHasCulator.git
cd CalHasCulator
```

Then just double-click the HTML file, or open it via `File > Open` in your browser.

## Tech Stack

- **HTML** – structure
- **CSS** – styling (embedded in the same file)
- **JavaScript** – calculator logic (embedded in the same file)

## Usage

Simply click the on-screen buttons (or use your keyboard, if supported) to enter numbers and operators, then press `=` to see the result.
